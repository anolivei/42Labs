## Starting ##

```bash
# Clone this project
$ git clone https://github.com/42sp/42labs-selection-process-v2-anolivei

# Access
$ cd 42labs-selection-process-v2-anolivei

# Run the project
$ bash run.sh

# The server will initialize in the <http://localhost:8080>
```
